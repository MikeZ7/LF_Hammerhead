# LF_Hammerhead

---

## Description

Project of line following robot HAMMERHEAD

## Hardware

* Arduino Nano Every
* QTR-8RC IR Reflactance sensor
* HC-06 Bluetooth module
* Pololu Magnetic encoders
* Pololu 1:10 Motors
* DRV8833 Adafruit Motor driver
